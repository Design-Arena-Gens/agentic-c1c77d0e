# Agente IA — Jurisprudência BR

Meta-buscador para jurisprudência, temas e decisões vinculantes nos sites oficiais (STF, STJ, TST, TRFs, TRTs e TJs selecionados).

- Next.js 14 (App Router)
- API serverless para buscar e fazer parse leve (quando possível)
- Fallback: link direto para a página de resultados do tribunal
